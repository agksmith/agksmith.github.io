Horizon - Archive

These old versions are still available in case you have serious problems getting the latest version to run. However, they are no longer supported.

Please note that installing an earlier version after running a later version can result in unexpected behaviour or crashes due to incompatibilities in the way program settings are saved in the Windows Registry. If all else fails, delete the saved registry settings from the Tools / Beta / Registry menu.

  • Version 0.12a (14-Jan-2019)
    • Fixed: Numerous bugs in the calculation engine and user interface.
    • Note: Previous versions contained a major bug that resulted in a half-cell offset with some SRTM data.
    • Added: Support for 1-arcsecond (30m) SRTM data (HGT and GeoTiff formats).
    • Added: Options for improving output image appearance for publication in monochrome (black-and-white).
    • Modified: Improved support for Stellarium landscapes.
    • Modified: Better performance and more compact data files.
    • Added: Windows Desktop integration.
  • Version 0.11b (01-Jan-2014)
    • Fixed: Crashes and unexpected behaviour if Windows is configured with localisation settings for most European countries.
    • Modified: Redesigned and improved the user interface for creating Stellarium landscapes.
    • Added: Site databases can be read from KML files.
    • Added: Buttons which load Google Earth (if installed) automatically to show locations.
    • Fixed: Offset errors between background and foreground when changing minimum display elevation.
    • Modified: Minor user interface improvements and tweaks.
